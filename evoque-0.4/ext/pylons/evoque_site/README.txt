This folder provides a sample of each file that would be affected by 
setting "evoque" as the default template engine in a Pylons application. 

The short list of these affected files is:

    development.ini
    evoque_site/config/environment.py
    evoque_site/lib/base.py
    # controller and template are only an example 
    evoque_site/controllers/hello.py # example controller only 
    evoque_site/templates/template.html

For details on how to use evoque with pylons see:

    http://evoque.gizmojo.org/ext/pylons/
